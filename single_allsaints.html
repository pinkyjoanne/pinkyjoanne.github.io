AllSaints LookBook UI

Branding, Wireframes, UI style guide, mockup, prototypes, speed test/errors

July, 2015 (two weeks)

AllSaints branding page on Amazon.com, associated with the Pay-with-Amazon widget on AllSaints.com, introduces the purchase-overall-look user experience. The online customer is able to choose an overall look at clothing suggested by the retailer, rather than selecting individual items, and to move to the item detail page for review. This project is being continued and moved to the next phase, thanks to the successful implementation of the user experience and positive customer feedback.
